# TECH ESPM - Pump it Up: Data Mining the Water Table:

O objetivo deste projeto é criar um modelo para a competição do Pump It Up:
https://www.drivendata.org/competitions/7/pump-it-up-data-mining-the-water-table/

Este projeto é composto pelos seguintes diretórios e arquivos: 

* **data**: possui os datasets iniciais do desafio e todos utilizados em conjunto, se há;
* **reports**: aqui é o lugar onde o markdown do R está;
* **scripts**: diretório que possui todos os scripts utilizados no desafio.
